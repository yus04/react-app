# react-app
React のサンプルアプリケーションを動かすためのコードです。このリポジトリ内のコードは、以下のコマンドを実行して作成されたものです。

```
npx create-react-app app
```

## 実行手順
サンプルアプリケーションへのディレクトリへ移動

```
cd app
```

必要なライブラリのインストール
```
npm install
```

サンプルアプリケーションの起動
```
npm start
```
